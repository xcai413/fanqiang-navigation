# iOS Shadowrocket（小火箭）使用教程

## 1. 获取 Shadowrocket

Shadowrocket 是 iOS 付费应用，需使用美区 Apple ID 在 App Store 下载。

- [获取免费美区 Apple ID](../free/apple-id.md)
- [购买成品美区账号 + Shadowrocket](https://xiaohuojian8.com/aff/5339.html)

## 2. 添加订阅

1. 打开 Shadowrocket
2. 点击右上角 **+** 号
3. 类型选择 **Subscribe**（订阅）
4. 粘贴 [订阅链接](../free/shadowrocket.md)
5. 点击 **Save** 保存

## 3. 连接

- 在首页选择节点
- 点击顶部的开关按钮连接
- 首次连接会提示添加 VPN 配置，点击 **Allow**

## 4. 全局路由设置

- 打开 Shadowrocket → 设置
- 建议将全局路由设置为 **代理** 或 **配置**
- 如只需翻墙，可选择 **配置** 模式
