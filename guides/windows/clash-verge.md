# Windows Clash Verge 使用教程

## 1. 下载客户端

下载 Clash Verge（开源免费）：

- [GitHub Releases](https://github.com/clash-verge-rev/clash-verge-rev/releases)
- 选择最新版本的 `.exe` 文件下载安装

## 2. 添加订阅

1. 打开 Clash Verge
2. 进入 **订阅** 页面
3. 点击 **添加**，输入订阅链接
4. 点击 **保存**，再点击 **更新**

## 3. 选择节点

- 进入 **代理** 页面
- 选择一个延迟低、速度快的节点
- 开启 **系统代理** 开关

## 4. 获取订阅链接

- [免费节点订阅](../free/clash-subscribe.md)
- 或使用付费机场提供的订阅链接
